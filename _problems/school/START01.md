<p>Write a program that accepts a number, <b>n</b>, and outputs the same.
</p>

<h3>Input</h3>
<p>The only line contains a single integer. </p>

<h3>Output</h3>
<p>Output the answer in a single line.</p>

<h3>Constraints</h3>
<ul>
<li><b>0 ≤ n ≤ 10<sup>5</sup></b></li>
</ul>


<h3>Sample Input</h3>
<pre>
123
</pre>
<h3>Sample Output</h3>
<pre>
123
</pre>
