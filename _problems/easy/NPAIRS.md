Given a string $S$ of length $N$ containing only numeric characters, find the number of **Nice Pairs**.

A **Nice Pair** is a pair of indices - $(i, j)$ such that $1 \leq i \lt j \leq N$ and $j - i = |S_j - S_i|$.
